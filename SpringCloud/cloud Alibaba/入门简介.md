# 为什么出现Cloud Alibaba

  ![image-20230602120501554](https://raw.githubusercontent.com/Janeonly300/codeImg/main/img/image-20230602120501554.png)在18年SpringCloud alibaba正式加入cloud官方孵化器.

有着以下特点

- 限流降级的接入和监控
- 服务注册发现
- 消息驱动
- 配置管理
- 分布式任务调度
